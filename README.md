Sebuah proyek tugas DDP (Dasar-Dasar Pemrograman) yang dibuat untuk mensimulasikan sistem perpustakaan digital sederhana berbasis bahasa C.  
Program ini menggunakan konsep **modular programming** dengan beberapa file terpisah (`library.c`, `peminjam.c`, `admin.c`, dan `main.c`) untuk memudahkan kerja tim dan pemeliharaan kode.

Program ini masih dalam tahap pengembangan, jadi mohon maklum jika adanya error atau bug yang menyebabkan kode tidak bekerja dengan selayaknya

---
## Fitur

Program ini memiliki fitur - fitur yang dibagi menjadi 2 berdasarkan role. Role tersebut adalah:

1. Role Admin

  Role admin memiliki fitur seperti:

- Menampilkan daftar buku
- Menambahkan buku
- Menghapus buku
- Mengecek denda keterlambatan secara global
- Menambahkan atau mengurangi stok buku
- Mencari buku berdasarkan judul
- Melihat daftar peminjaman buku

2. Role User

  Role user memiliki fitur seperti:

- Meminjam buku
- Mencari buku berdasarkan judul
- Mengembalikan buku
- Melihat status pinjaman
- Melaporkan kehilangan buku
